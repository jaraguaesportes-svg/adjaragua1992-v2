# adjaragua1992 V2.0 — Esqueleto inicial

MVP inicial em Next.js + Firebase para começar a aplicação.

## Rodar localmente

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Primeiras rotas

- `/` — página inicial pública
- `/login` — login administrativo
- `/admin` — painel administrativo
- `/admin/people` — CRUD inicial de pessoas
- `/admin/games` — CRUD inicial de jogos
